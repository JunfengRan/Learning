# Learning

Agent 相关学习与见闻记录。本仓库按主题分类整理笔记、论文摘要、博客摘录与前沿工具调研，方便检索与持续更新。

## 目录结构

| 目录 | 说明 |
|------|------|
| [frontier-apps/](./frontier-apps/) | 前沿 Agent 应用与开源项目（OpenCode、Codex、OpenClaw 等） |
| [model-training/](./model-training/) | 最新模型训练实践与工程经验 |
| [papers/](./papers/) | 有意思的论文阅读笔记 |
| [personal-blogs/](./personal-blogs/) | 有意思的个人博客摘录 |
| [company-blogs/](./company-blogs/) | 各家 AI 公司 Research Blog 摘录 |

## 使用方式

- 每个目录下有独立的 `README.md` 作为索引，新增内容时同步更新索引。
- 单篇笔记建议命名：`YYYY-MM-DD-简短标题.md`，便于按时间浏览。
- 论文、博客摘录用 Markdown；超长应用技术深挖可另附同名 `.html`（摘要 `.md` 链到完整版）。
- 摘录外部内容时注明来源链接与阅读日期。
